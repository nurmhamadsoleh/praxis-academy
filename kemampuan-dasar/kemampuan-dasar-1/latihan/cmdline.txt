date --> menampilkan Date & Time Zone untuk saat ini/ Real Time
mkdir (nama folder)--> untuk membuat direktori / folder baru
cd --> digunakan untuk perpindah direktori
ls --> untuk menapilkan daftar nama file atau folder/direktori 
pwd --> diguanakan untuk melihat/mengecek posisi saat ini
code . --> untuk membuka teks editor seperti visual studio code
rmdir --> untuk menghapus direktori/folder
rm --> untuk mengapus folder/direktori sekaligus isi file yang ada di dalam derektori/folder
ping (dns google) --> Melakuka cek koneksi internet anda ke server stabil atau tidak
systeminfo --> untuk memberikan informasi detail dari system operasi & juga informasi detail terkait hardware yang di gunakan.
